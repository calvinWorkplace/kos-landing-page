# KostExecutivePloso — Landing Page

Landing page modern untuk **KostExecutivePloso**, kos-kosan executive di Jl. Ploso 1 No.22, Surabaya.

## Tech Stack

| Layer | Library / Tool |
|---|---|
| Framework | Next.js 16 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS v4 |
| Icons | Lucide React |
| Dark Mode | next-themes |
| i18n | React Context (ID / EN) |

## Fitur

- **Hero Section** — background full-screen, CTA WhatsApp, statistik kos
- **Fasilitas** — 8 fasilitas utama dengan ikon
- **Katalog Kamar** — 4 tipe kamar (A–D) dengan harga, status, dan tombol WA
- **Lokasi Strategis** — 2 kategori: kampus terdekat & pusat belanja
- **Testimoni** — 4 ulasan asli dari Google Maps (rating 4.1 / 90 ulasan)
- **FAQ** — 6 pertanyaan umum dengan accordion
- **Dark Mode** — toggle terang/gelap
- **Bilingual** — switch Bahasa Indonesia / English

## Menjalankan di Lokal

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000).
