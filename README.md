## ✨ Features

| Feature | Description |
|---|---|
| 🦸 **Hero Section** | Full-screen background with WhatsApp CTA and live statistics |
| 🛁 **Facilities** | 8 premium amenities showcased with icons |
| 🛏️ **Room Catalog** | 4 room types (A–D) with pricing, availability, and direct WA booking |
| 📍 **Strategic Location** | Nearby campuses and shopping centers highlighted |
| 💬 **Testimonials** | 4 verified Google Maps reviews (4.1★ from 90+ ratings) |
| ❓ **FAQ** | 6 common questions with smooth accordion animation |
| 🌙 **Dark Mode** | Seamless light/dark toggle powered by `next-themes` |
| 🌐 **Bilingual** | Full Bahasa Indonesia / English language switcher |

## 🚀 Getting Started

### Prerequisites

- Node.js `>=18.x`
- npm or yarn

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/kos-landing.git
cd kos-landing

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
